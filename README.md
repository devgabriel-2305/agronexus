
# 🌾 AgroNexus

Uma landing page institucional fictícia voltada para o setor agropecuário, desenvolvida com foco em organização visual, responsividade e experiência do usuário.

---

## 🚀 Sobre o Projeto

O **AgroNexus** apresenta uma empresa do agronegócio de forma moderna e estruturada, destacando:

- 🌱 Identidade institucional
- 🚜 Serviços oferecidos
- 🌽 Produtos
- 📍 Contato com localização
- 📱 Navegação responsiva

O projeto foi desenvolvido como prática de front-end, explorando layout, animações e manipulação de DOM.

---

## 🛠 Tecnologias

- HTML5
- CSS3
- Tailwind CSS
- JavaScript

---

## ✨ Destaques Técnicos

- Scroll suave entre seções  
- Menu mobile interativo  
- Animações com `IntersectionObserver`  
- Verificação automática de horário de funcionamento  
- Layout estruturado com Grid e Flexbox  

---

## 🎯 Objetivo

Aprimorar habilidades em desenvolvimento front-end, organização de código e criação de interfaces responsivas.
